## Format ADR

Chaque ADR suit le template standard :

1. **Title** - Titre descriptif
2. **Status** - Proposed / Accepted / Deprecated / Superseded
3. **Context** - Le problème ou la situation
4. **Decision** - La décision prise
5. **Consequences** - Les impacts positifs et négatifs
